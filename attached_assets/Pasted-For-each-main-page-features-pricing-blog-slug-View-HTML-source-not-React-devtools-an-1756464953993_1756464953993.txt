5. JSON-LD structured data
In HTML <head> find a single <script type="application/ld+json">.

Paste its contents into Google’s Rich Results Test – should return 0 errors / 0 warnings.

Organization schema must include: "@type":"Organization", "url", "logo", "name".
WebSite schema must include "@type":"WebSite", "potentialAction" (SearchAction).
Blog/Article pages must include "headline", "datePublished", "author".

6. Prerendering / Static HTML
In Chrome DevTools → Network Conditions, set user-agent to Googlebot and reload /.

Initial HTML must already contain the main content (not just a <div id="root"></div>).

Response headers should include x-prerender or similar flag your script sets.

Disable JavaScript (DevTools > Settings > Debugger) and reload. Page should still show meaningful content and links.

Time-to-First-Byte (TTFB) ≤ 600 ms on first load.
